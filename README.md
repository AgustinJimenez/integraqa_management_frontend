### INSTALL DEPENDENCIES

```
    yarn
```

### CREATE .ENV

```
    cp env.example.json env.json
```

### RUN SERVER ON DEV-MODE

```
    yarn start
```

### EXPORT TO PRODUCTION

```
    yarn build
```
